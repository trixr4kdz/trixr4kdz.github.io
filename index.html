My Projects

GRNmap

GRNmap is a MATLAB bioinformatics software that estimates the parameters (production/repression threshold, network weights, production rates) of medium-scale gene regulatory networks and allows forward simulation on the network. The program outputs individual gene plots in addition to the estimated parameters after running calculations on Excel spreadsheet data inputs.

WebRTC Video Chat

A simple demonstration of what the two-way video chat using WebRTC looks like on a browser. Signaling server is written in NodeJS and is hosted on AWS EC2. This application can be run using Chrome or Firefox browsers. To use, simply open two separate web browsers and enter the IP address https://34.210.146.191:3000/ to access the application.
